
# Design system

Rollup plugin for design system (storybook).

## Demo
[https://najwer23.github.io/najwer23storybook/](https://najwer23.github.io/najwer23storybook/)

## Install

### yarn
```
yarn add najwer23storybook --dev
```

### npm
```
npm i najwer23storybook
```

## Requirements
```
"react": "^18.2.0",
"react-dom": "^18.2.0"
"typescript": "^5.3.3"
"storybook": "^7.6.6",
```
