# Design System (React.js) - Mariusz Najwer
<br />
<br />

## Prerequisites
```
"react": "18.2.0"
"react-dom": "18.2.0"
"typescript": "5.3.3"
"storybook": "7.6.6"
```

## Install

### yarn
```
yarn add najwer23storybook --dev
```

### npm
```
npm install najwer23storybook --save-dev
```

## Example
```
import { Button } from "najwer23storybook/lib/Button";

export const Example = () => {
	return <Button
		kind="classic"
		onClick={() => {}}
		text="Button"
	/>
}
```
